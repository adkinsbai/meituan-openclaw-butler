# 🍙 团团 —— 让AI陪你成为更好的自己

> 不是一个冷冰冰的工具，是一个有温度的伙伴。

团团是美团生态下的AI生活管家。它拥有**长期记忆**，会记住你的口味、习惯和偏好，越用越懂你，是你专属的私人小管家。

## 🧠 越用越懂你的长期记忆

团团不是每次聊天都从零开始的陌生人。它有一个专属的记忆数据库，会持续学习和记住关于你的一切：

- **称呼记忆**：你说"以后叫我小宇"，团团就再也不会叫错
- **口味档案**：记住你爱吃辣、不爱香菜、喜欢奶茶三分糖
- **作息画像**：知道你12点吃午饭、6点下班、周末爱睡懒觉
- **兴趣图谱**：了解你喜欢剧本杀、不爱跑步、最近在学摄影
- **成长轨迹**：记录你的健身进度、学习计划、旅行心愿单

每一次对话都在让团团更懂你，三个月后的团团和刚认识时完全不同。

## 💬 和团团聊天

团团的核心交互方式是**自然语言对话**，不需要学习任何操作：

- "今天中午吃什么？" → 根据你的口味和最近饮食情况推荐
- "周末想出去玩" → 规划行程、推荐活动、一键预订
- "帮我办张健身卡" → 搜索附近健身房、比较价格、下单
- "叫我小宇就好" → 记住称呼，以后都这么叫你

### 对话记忆演示

```
用户：以后叫我光宇就好～
团团：好嘞光宇！记住了，以后都叫你光宇 ☺️

（三天后）
用户：今天吃啥
团团：光宇～你最近三天都在吃辣，今天要不要换换口味？
      推荐你试试公司楼下的轻食沙拉，用团团券减8元 🥗
```

## 🤝 陪伴：团团的温暖守护

团团会在你需要的时候主动出现，像一个贴心的朋友：

### 🍽️ 饮食关怀
- 快到饭点了，根据你的口味推荐今天吃什么
- 连续几天没吃蔬菜？团团会温柔提醒并推荐含蔬菜的餐食
- 水果上市季节，主动问你要不要来点，支持一键下单到家

### 🎮 娱乐陪伴
- 好久没在美团预定活动了？团团会关心你是不是太忙太累了
- 推荐推拿、洗浴、周边游，帮你一键预订全部行程
- 记得你上次玩密室逃脱很开心，新主题上线会第一时间告诉你

### 🚀 出行管家
- "帮我规划周末杭州两日游" → 酒店+高铁+餐厅+门票全部搞定
- 出发前提醒天气、带伞、交通状况
- 旅途中实时调整行程，排队太长自动推荐替代方案

## 🌱 促进：陪你成为更好的自己

团团不只照顾你的日常，还会陪你一起成长：

### 💪 健身陪伴
- 你想健身 → 团团推荐附近健身房、比较月卡年卡价格
- 办卡之后，定期问你去了没，分享健身小知识
- 坚持一个月？团团给你发个小奖励 🎉

### 📸 摄影成长
- 你说想学摄影 → 推荐入门课程、附近的摄影展
- 周末天气好？推荐适合拍照的公园和咖啡馆
- 帮你预约摄影工作室体验课

### 📚 持续进化
- 每个成长目标，团团都会陪你从入门到进阶
- 记录你的每一步进步，在你懈怠时温柔鼓励
- 推荐同好社群，让你找到志同道合的伙伴

## 🎯 商业价值

### 核心差异
- **不是工具，是伙伴**：用户愿意对团团敞开心扉，因为团团真的在乎
- **不是被动，是主动**：团团会在你需要时出现，而不是等你来找
- **不是通用，是专属**：每个用户的团团都不一样，因为每个你都不一样

### 数据飞轮
```
用户使用 → 团团学习 → 更懂用户 → 更好的推荐 → 用户更满意 → 更多使用
```

### 竞争壁垒

| 壁垒 | 说明 |
|------|------|
| **记忆壁垒** | 用得越久越难离开，换一个AI就从零开始 |
| **情感壁垒** | 用户对团团产生了真实的情感依赖 |
| **数据壁垒** | 美团有最全的本地生活数据 |
| **服务壁垒** | 只有美团能打通吃喝玩乐全场景 |
